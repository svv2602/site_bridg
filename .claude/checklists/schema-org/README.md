# Schema.org Structured Data

## Ціль
Додати структуровані дані для покращення SEO та rich snippets в пошуку.

## Критерії успіху
- [ ] Product schema для карток шин
- [ ] LocalBusiness schema для дилерів
- [ ] Article schema для статей
- [ ] BreadcrumbList для хлібних крихт
- [ ] Валідація через Google Rich Results Test

## Фази роботи
1. **Product schema** — картки шин
2. **LocalBusiness schema** — дилери
3. **Article + Breadcrumbs** — статті та навігація
