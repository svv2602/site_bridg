# Google Maps інтеграція — карта дилерів

## Ціль
Реалізувати інтерактивну карту дилерів з маркерами, фільтрами та інформаційними вікнами.

## Критерії успіху
- [ ] Карта відображає всіх дилерів з бази даних
- [ ] Фільтрація за містом/областю працює
- [ ] Фільтрація за типом точки працює
- [ ] При кліку на маркер показується інфо-вікно з контактами
- [ ] Кнопка "Маршрут" відкриває Google Maps з напрямком
- [ ] Карта адаптивна (mobile/tablet/desktop)

## Фази роботи
1. **Підготовка** — налаштування Google Maps API, env змінні
2. **Базова карта** — відображення карти та маркерів
3. **Інтерактивність** — інфо-вікна, фільтри, пошук
4. **Фінальна інтеграція** — заміна placeholder, тестування

## Правила перевикористання коду

### Де шукати:
```
frontend/src/app/dealers/page.tsx    # Поточна сторінка з placeholder
frontend/src/lib/data.ts             # Тип Dealer з координатами
frontend/src/lib/api/dealers.ts      # API функції для дилерів
```

### Чекліст перед написанням коду:
- [ ] Вивчив структуру dealers/page.tsx?
- [ ] Перевірив тип Dealer в data.ts (latitude, longitude)?
- [ ] Дотримуюся патернів стилів проекту?

## Технічні рішення

### Варіанти реалізації:
1. **@react-google-maps/api** — офіційна React обгортка (рекомендовано)
2. **react-leaflet + OpenStreetMap** — безкоштовна альтернатива
3. **Google Maps Embed API** — простіше, але менш гнучке

### Env змінні:
```env
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_api_key
```
