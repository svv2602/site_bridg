## Критические проблемы (P0)

### 1. Нефункциональные кнопки и ссылки
- **Главная:** кнопки "Перейти", "Детальніше", "Знайти дилера", "Читати статтю" не работают
- **Контент:** ~15 кнопок без onClick handlers или href
- **QuickSearchForm:** форма поиска не отправляет данные

### 2. Отсутствие API интеграции
- `/advice` и `/technology` используют mock-данные напрямую вместо API
- `/dealers` игнорирует API слой с CMS fallback
- Данные захардкожены вместо загрузки из Payload CMS

### 3. Security (Админка)
- Default credentials в коде (`admin:bridgestone2026`)
- Нет CSRF защиты для API вызовов
- Нет rate limiting

### 4. Формы без функциональности
- Контактная форма (`/contacts`) не имеет submit handler
- Форма поиска шин не переходит на результаты

### i18n (100% hardcoded текст)

Весь UI текст захардкожен на украинском языке:
- ~200+ строк в компонентах
- Невозможно редактировать контент без деплоя
- Подготовка к мультиязычности требует полного рефакторинга

**Рекомендация:** Создать `lib/i18n/uk.ts` и использовать next-intl
