### 6.4 Переключатель Car/Truck в хедері (optional)

- [ ] Оцінити необхідність (чи є достатньо комерційних шин?)
- [ ] Якщо потрібен: додати toggle в хедер
- [ ] Змінювати контент на основі вибору
- [ ] Зберігати вибір в localStorage

**Файли:** `frontend/src/components/MainHeader.tsx`

**Реалізація (якщо потрібна):**
```tsx
function VehicleTypeToggle() {
  const [type, setType] = useState<'car' | 'truck'>('car');

  return (
    <div className="flex rounded-full bg-stone-800 p-1">
      <button
        onClick={() => setType('car')}
        className={`px-4 py-1 rounded-full text-sm ${
          type === 'car' ? 'bg-white text-stone-900' : 'text-stone-300'
        }`}
      >
        Легкові
      </button>
      <button
        onClick={() => setType('truck')}
        className={`px-4 py-1 rounded-full text-sm ${
          type === 'truck' ? 'bg-white text-stone-900' : 'text-stone-300'
        }`}
      >
        Комерційні
      </button>
    </div>
  );
}
```

**Нотатки:** Можливо не потрібен, якщо мало комерційних шин в каталозі
